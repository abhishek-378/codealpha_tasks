# 🎵 Music Player

A modern and responsive Music Player built using **HTML, CSS, and JavaScript** as part of the **CodeAlpha Frontend Development Internship**.

## ✨ Features

- 🎵 Play, Pause, Next & Previous Controls
- 🖼️ Song Cover Display
- 📊 Interactive Progress Bar
- 🔊 Volume Control
- 📱 Fully Responsive Design
- 🎧 Playlist Support

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
