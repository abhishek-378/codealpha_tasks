# 📷 Image Gallery

A modern and responsive Image Gallery built using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## ✨ Features

- Responsive design
- Category-based image organization
- Smooth hover effects
- Clean and modern user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
